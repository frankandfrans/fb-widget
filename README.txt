Frank and Fran's Fishing Report BASE64 APP version (FINAL FIX → Yodeck Android Compatible)

This version downloads Facebook images → encodes them in BASE64 → embeds directly in HTML → NO external resources → WILL render on Android Yodeck players.

How to run:

1. Upload this to GitHub → connect to Render.com → deploy Web Service.
2. Set ACCESS_TOKEN as environment variable with your Facebook Page token.
3. Access the app URL → will show Fishing Report with images → no black screen guaranteed.
4. Add URL to Yodeck as Web Page → done.
